# Health-Tracker
## How to configuration?
First step:\
Go to Health_Tracker -> src -> Util and open ImageUtil.java. Then set uploadLocationProfil and uploadLocationFood to your own directory.\
\
Second step:\
Import health_tracker.sql to your mysql server.

#### That's all :)
